## Joins & Fan-Out

| Feature | Status | Notes |
|---|---|---|
| Dimensional lookups (M:1 join) | ✅ | YAML joins with relationship declarations |
| Multi-fact joins (M:M / fan-out) | ✅ | Automatic CTE dedup via symmetric aggregates |
| Self-joins | ✅ | Alias property |
| Chained joins (A→B→C) | ✅ | Fan-out protection propagates |
| Cross joins | ❌ | Not a valid join type |
| Ad-hoc joins (user-defined at query time) | ❌ | Must be pre-defined in YAML |

## Aggregations

| Feature | Status | Notes |
|---|---|---|
| Basic aggregates (sum, count, avg, min, max) | ✅ | |
| Count distinct | ✅ | |
| Median / percentile | ✅ | Configurable N |
| Approximate count distinct (HLL) | ❌ | No native type |
| Conditional aggregates (filtered metrics) | ✅ | `filters:` on metrics |
| Derived metrics / ratios (metric ÷ metric) | ✅ | `type: number` referencing other metrics |

## Level of Detail (mixed-grain analysis)

| Feature | Status | Notes |
|---|---|---|
| Fixed-grain metric (e.g. total revenue alongside per-order rows) | ❌ | No FIXED equivalent; must pre-compute in dbt |
| Exclude dimensions from aggregation | ❌ | No EXCLUDE equivalent |
| Include extra dimensions in aggregation | ❌ | No INCLUDE equivalent |
| Percent of total (as semantic metric) | 🟡 | Experimental `type: percent_of_total` |
| Metric at coarser grain for benchmarks / ratios | ❌ | Can't express "company avg" alongside "rep total" |

## Time Intelligence

| Feature | Status | Notes |
|---|---|---|
| Date granularity switching (zoom) | ✅ | Auto-generated DAY/WEEK/MONTH/QUARTER/YEAR |
| Period-over-period (single comparison) | 🟡 | Beta; one comparison per chart |
| Multiple PoP comparisons on one chart | 🟡 | Workaround via parameters; not native |
| Rolling/trailing window (e.g. 7-day avg) | 🟡 | Table calc only; row-based, breaks with date gaps |
| Cumulative / running total | 🟡 | Experimental metric type |
| Semi-additive metrics (balance at end of period) | ❌ | Must pre-compute in dbt |
| Date spine / fill missing dates | ❌ | Must use `dbt_utils.date_spine` |
| Fiscal calendar | 🟡 | Custom granularities (Early Access); no project-wide offset |

## Merged Queries / Blended Data

| Feature | Status | Notes |
