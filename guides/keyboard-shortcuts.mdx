Use keyboard shortcuts to navigate and work more efficiently in Lightdash.

## Global shortcuts

These shortcuts work throughout the Lightdash application.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd/Ctrl + K` | Global search | Open the global search to quickly find charts, dashboards, and explores |

## Explorer shortcuts

These shortcuts are available when working in the Explorer view.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd/Ctrl + Enter` | Run query | Execute the current query to see results |
| `Cmd/Ctrl + Shift + Backspace` | Clear explore | Reset all selections and filters in the current explore |

## Tips

- On Mac, use `Cmd` for shortcuts
- On Windows and Linux, use `Ctrl` for shortcuts
- Keyboard shortcuts help you work faster without reaching for your mouse
