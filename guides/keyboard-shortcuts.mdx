Use keyboard shortcuts to navigate and work more efficiently in Lightdash.

<Tip>
  **How to use shortcuts**

  - On Mac, use `Cmd` for shortcuts
  - On Windows and Linux, use `Ctrl` for shortcuts
</Tip>

## Global shortcuts

This shortcut works throughout the Lightdash application.

| Shortcut     | Action        | Description                                                             |
| ------------ | ------------- | ----------------------------------------------------------------------- |
| Cmd/Ctrl + K | Global search | Open the global search to quickly find charts, dashboards, and explores |

## Query from tables shortcuts

These shortcuts are available when creating a new Query from tables or editing a saved chart.

| Shortcut                             | Action          | Description                                                              |
| ------------------------------------ | --------------- | ------------------------------------------------------------------------ |
| Cmd + Return<br />Ctrl + Enter       | Run query       | Execute the current query to see results                                 |
| Cmd + Option + k<br />Ctrl + Alt + k | Clear explore   | Reset all selections and filters in the current explore                  |
| Cmd/Ctrl + c                         | Copy table cell | When a table cell is selected this copies the contents to your clipboard |
