Flat tables list every combination as a row. Pivoting flips one dimension into column headers, turning a long list into a compact grid humans can actually scan.

<Info>**Status:** ✅ Fully supported.</Info>

## The question

> "Show me revenue by product category, with each month as a column."

## Why it matters

A table with 3 products and 4 months produces 12 rows in flat format. Pivoted, it's a tidy 3-by-4 grid where trends jump off the page. Spreadsheet users expect this layout, and Lightdash delivers it without exporting to Excel.

## Example

Quarterly revenue for an online store's product categories.

### Source data

| order_month | category | order_total |
|-------------|----------|-------------|
| 2025-01 | Electronics | 349.00 |
| 2025-01 | Apparel | 89.00 |
| 2025-02 | Electronics | 478.00 |
| 2025-02 | Home | 215.00 |
| 2025-03 | Apparel | 132.00 |
| 2025-03 | Electronics | 599.00 |
| 2025-03 | Home | 167.00 |

### YAML

```yaml
version: 2
models:
  - name: orders
    columns:
      - name: order_month
        meta:
          dimension:
            type: date
      - name: category
        meta:
          dimension:
            type: string
      - name: order_total
        meta:
          metrics:
            total_revenue:
              type: sum
              format: "usd"
```

### Result (flat)

| category | order_month | total_revenue |
|----------|-------------|---------------|
| Electronics | 2025-01 | $42,300 |
| Electronics | 2025-02 | $51,800 |
| Electronics | 2025-03 | $48,100 |
| Apparel | 2025-01 | $18,400 |
| Apparel | 2025-02 | $21,200 |
| Apparel | 2025-03 | $19,700 |
| Home | 2025-01 | $12,100 |
| Home | 2025-02 | $14,600 |
| Home | 2025-03 | $13,900 |

### Result (pivoted on order_month)

| category | 2025-01 | 2025-02 | 2025-03 |
|----------|---------|---------|---------|
| Electronics | $42,300 | $51,800 | $48,100 |
| Apparel | $18,400 | $21,200 | $19,700 |
| Home | $12,100 | $14,600 | $13,900 |

## How to set it up

1. Run a query with two dimensions (e.g. `category` and `order_month`) and a metric.
2. In the results table, click the **Pivot** button on the dimension you want as column headers.
3. The table restructures instantly — no re-query needed.
4. Toggle **Show subtotals** for row and column totals.

## Tips

- You can pivot up to 3 dimensions and 200 columns.
- Pivoting works on the results table and in saved charts on dashboards.
- If you have too many unique values in the pivoted dimension, filter first to keep the grid readable.
